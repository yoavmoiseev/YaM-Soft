📚 Что такое School Auto Chat?

School Auto Chat — это полностью офлайн-платформа для общения в классе. Работает на компьютере учителя, ученики подключаются через браузер по локальной сети (Wi-Fi или кабель) — без интернета и без установки какого-либо ПО.

ФункцияОписание
💬 Чат в реальном времениОбщий чат класса, виден всем участникам
🔒 Личные сообщенияОтправка сообщения конкретному человеку
🤖 ТичерБотПоиск по локальным файлам базы знаний
👨‍🏫 Роль администратораУчитель получает полный контроль модерации
🌐 Работает офлайнИнтернет не требуется ни на каком этапе
📱 Адаптивный дизайнРаботает на ноутбуках, планшетах и телефонах

🚀 Быстрый старт: Учитель

Онлайн-версия (Python установлен на компьютере)

  1. Откройте папку Chat на своём компьютере.
  2. Дважды щёлкните run.bat — сервер запустится. В консольном окне появится адрес.
  3. Откройте браузер и перейдите по адресу http://localhost:5002.
  4. Зарегистрируйтесь с любым именем и паролем администратора (по умолчанию: teacher123, задаётся в server.py). Вы автоматически получите права администратора.
  5. Сообщите ученикам ваш IP-адрес в локальной сети (показан в консоли, например: http://192.168.1.5:5002).
✅ Совет: Когда Windows Firewall спросит разрешение — нажмите Разрешить доступ. Это необходимо для подключения учеников.

Офлайн-версия (Python не нужен)

  1. Распакуйте ChatApp-Offline.zip в любую папку (USB-флешка, Рабочий стол и т.д.).
  2. Дважды щёлкните START_CHAT.bat. Браузер откроется автоматически.
  3. В консоли будет показан адрес сети — передайте его ученикам.
ℹ️ Офлайн-версия включает Python 3.11 и все зависимости. Устанавливать ничего не нужно.

🎒 Быстрый старт: Ученик

  1. Убедитесь, что вы подключены к той же Wi-Fi-сети или локальной сети, что и компьютер учителя.
  2. Откройте любой браузер и введите адрес, который дал учитель, например: http://192.168.1.5:5002.
  3. Введите своё имя (или логин, если уже зарегистрировались) и пароль.
  4. Вы в чате класса! Напишите сообщение внизу и нажмите Enter или кнопку отправки.
⚠️ Ничего устанавливать не нужно — достаточно обычного браузера (Chrome, Firefox, Edge, Яндекс.Браузер).

💬 Функции чата

Общий чат

Введите сообщение в поле внизу и нажмите Enter. Его увидят все участники.

Личные сообщения

Нажмите на имя пользователя в панели Онлайн-пользователи справа — введите сообщение. Его получит только этот человек.

Уведомления

При получении личного сообщения на имени отправителя появится значок. Заголовок вкладки браузера тоже обновится.

История сообщений

При входе отображаются последние 50 сообщений (настраивается в server.py).

🤖 Руководство по ТичерБоту

ТичерБот ищет в файлах knowledge_base/ и возвращает нужный фрагмент. Отправьте сообщение, начинающееся с @bot:

@bot Брандмауэр
@bot IP адрес
@bot ОЗУ

Бот ответит в общем чате найденным определением или фрагментом текста из базы знаний.

Как работает поиск

ПараметрОписание
РегистрБез учёта регистра (firewall = Firewall = FIREWALL)
Тип совпаденияПоиск подстроки — найдёт слово в любом контексте
Результат~300 символов вокруг первого вхождения
Нет совпаденийБот предлагает похожие слова из базы
Несколько файловПоиск по всем .txt и .html файлам в knowledge_base/

Пример взаимодействия

Ученик: @bot протокол
ТичерБот: Найдено в IT_Glossary.txt:
  "Протокол — набор правил, регулирующих обмен данными
   между устройствами в сети. Примеры: TCP/IP, HTTP, FTP."
✅ Рекомендация: Пишите каждый термин один раз в файле. При нескольких вхождениях возвращается только первое.

💾 Офлайн-версия (USB / локальная сеть)

Пакет ChatApp-Offline.zip — полностью автономный: Python, Flask и все зависимости включены. Интернет и установка не требуются.

Структура ZIP-архива

ChatApp-Offline/
├── START_CHAT.bat       ← дважды щёлкните для запуска
├── launcher_offline.py  ← определяет порт, открывает браузер
├── server.py            ← основной сервер чата
├── python/              ← встроенный Python 3.11
├── static/              ← CSS, JS, изображения
├── templates/           ← HTML-шаблоны
└── knowledge_base/      ← ваши файлы .txt / .html

Инструкция

  1. Распакуйте ChatApp-Offline.zip в любую локальную папку или на USB-флешку.
  2. Дважды щёлкните START_CHAT.bat.
  3. Браузер откроется автоматически на http://localhost:5002.
  4. В консоли также будет адрес локальной сети — сообщите его ученикам.
ℹ️ Запускатель автоматически пробует порты 5002, 5003, 5001, 8000, 8888, 5000 пока не найдёт свободный.
⚠️ Не перемещайте и не переименовывайте файлы внутри распакованной папки. Сервер использует относительные пути.

📝 Формат базы знаний

Поместите файлы .txt или .html в папку knowledge_base/. ТичерБот читает их при запуске.

Формат TXT (рекомендуется)

1. ОЗУ (Оперативная память) — Временная память компьютера, используемая при работе программ.
2. Процессор (CPU) — Основной вычислительный чип, выполняющий инструкции программ.
3. Брандмауэр — Система сетевой безопасности, контролирующая входящий и исходящий трафик.

Формат HTML

Используйте только простые теги. Бот извлекает видимый текст, игнорируя CSS и скрипты.

<p>Брандмауэр — система фильтрации сетевого трафика.</p>
<p>IP-адрес — логический адрес сетевого устройства.</p>

Рекомендации

  • Сохраняйте файлы в кодировке UTF-8 (особенно важно для кириллицы и иврита).
  • Пишите каждый термин один раз в файле.
  • Используйте формат Термин — Определение.
  • Заканчивайте предложения точкой для корректного ограничения контекста.

🔧 Решение проблем

ПроблемаРешение
Ученики не могут подключитьсяУбедитесь, что брандмауэр Windows разрешил доступ серверу. Все устройства должны быть в одной сети. Проверьте командой ping <IP учителя>.
Браузер не открылся автоматическиОткройте вручную: http://localhost:5002 (офлайн) или http://localhost:5002 (онлайн).
Сервер запускается, но есть ошибкиПроверьте консольное окно. Чаще всего причина — занятый порт. Запускатель автоматически найдёт свободный.
ТичерБот отвечает «не найдено»Убедитесь, что в папке knowledge_base/ есть файлы .txt или .html, сохранённые в UTF-8.
Кракозябры / проблемы с кодировкойПересохраните файлы базы знаний в UTF-8 (без BOM для .txt).
Не могу войти — неверный парольПользователи хранятся в users_db.json. Попросите администратора удалить и пересоздать аккаунт.
