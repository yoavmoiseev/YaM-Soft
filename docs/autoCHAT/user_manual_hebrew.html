📚 מה זה School Auto Chat?

School Auto Chat הוא פלטפורמת תקשורת כיתתית אופליין מלאה. הוא רץ על המחשב של המורה, והתלמידים מתחברים דרך הדפדפן ברשת המקומית (Wi-Fi או כבל) — ללא אינטרנט וללא צורך בהתקנה.

תכונהפירוט
💬 צ'אט בזמן אמתצ'אט ציבורי של הכיתה, גלוי לכולם
🔒 הודעות פרטיותשליחת הודעה לאדם ספציפי
🤖 TeacherBotבוט חכם שמחפש בקבצי בסיס הידע המקומי
👨‍🏫 תפקיד מנהלהמורה מקבל כלי פיקוח מלאים
🌐 עובד אופלייןלא נדרש אינטרנט בשום שלב
📱 עיצוב רספונסיביעובד על מחשבים ניידים, טאבלטים וטלפונים

🚀 התחלה מהירה: מורה

גרסה מקוונת (Python מותקן)

  1. פתח את תיקיית Chat במחשב.
  2. לחץ פעמיים על run.bat — השרת יתחיל. בחלון הקונסול יופיע הכתובת.
  3. פתח דפדפן וגלוש אל http://localhost:5002.
  4. הירשם עם כל שם משתמש והסיסמת מנהל (ברירת מחדל: teacher123, מוגדרת ב-server.py). תקבל אוטומטית הרשאות מנהל.
  5. שתף עם התלמידים את כתובת ה-IP המקומית שלך (מוצגת בקונסול, לדוגמה: http://192.168.1.5:5002).
✅ טיפ: כאשר חומת האש של Windows תשאל — לחץ אפשר גישה. זה הכרחי לחיבור התלמידים.

גרסה אופליין (ללא Python)

  1. חלץ את ChatApp-Offline.zip לכל תיקייה (USB, שולחן עבודה וכו').
  2. לחץ פעמיים על START_CHAT.bat. הדפדפן יפתח אוטומטית.
  3. בקונסול תופיע כתובת הרשת — שתף אותה עם התלמידים.
ℹ️ הגרסה האופליין כוללת Python 3.11 וכל התלויות. אין צורך להתקין דבר.

🎒 התחלה מהירה: תלמיד

  1. ודא שאתה מחובר לאותה רשת Wi-Fi או LAN כמו מחשב המורה.
  2. פתח כל דפדפן והקלד את הכתובת שהמורה נתן, לדוגמה: http://192.168.1.5:5002.
  3. הכנס את שמך (או שם המשתמש הרשום) והסיסמה.
  4. אתה בצ'אט הכיתה! כתוב הודעה בתחתית ולחץ Enter או כפתור השליחה.
⚠️ אין צורך להתקין כלום. כל דפדפן רגיל מתאים (Chrome, Firefox, Edge).

💬 תכונות הצ'אט

צ'אט ציבורי

כתוב הודעה בשדה הטקסט התחתון ולחץ Enter. כל המשתתפים יראו אותה.

הודעות פרטיות

לחץ על שם משתמש ברשימת המשתמשים המחוברים בצד ימין — כתוב הודעה. רק אותו אדם יקבל אותה.

התראות

כאשר תקבל הודעה פרטית, תופיע תגית על שם השולח. כותרת לשונית הדפדפן תתעדכן גם כן.

היסטוריית הודעות

עם הכניסה מוצגות 50 ההודעות האחרונות (ניתן לשינוי ב-server.py).

🤖 מדריך TeacherBot

TeacherBot מחפש בקבצי knowledge_base/ ומחזיר את הקטע הרלוונטי. שלח הודעה שמתחילה ב-@bot:

@bot חומת אש
@bot כתובת IP
@bot זיכרון RAM

הבוט יענה בצ'אט הציבורי עם ההגדרה או הקטע שנמצא בבסיס הידע.

💾 גרסה אופליין (USB / רשת)

החבילה ChatApp-Offline.zip עצמאית לחלוטין: Python, Flask וכל התלויות כלולות. אינטרנט והתקנה אינם נדרשים.

מבנה ה-ZIP

ChatApp-Offline/
├── START_CHAT.bat       ← לחץ פעמיים להפעלה
├── launcher_offline.py  ← מזהה פורט ופותח דפדפן
├── server.py            ← שרת הצ'אט הראשי
├── python/              ← Python 3.11 מוטמע
├── static/              ← CSS, JS, תמונות
├── templates/           ← תבניות HTML
└── knowledge_base/      ← קבצי .txt / .html שלך

הוראות

  1. חלץ את ChatApp-Offline.zip לכל תיקייה מקומית או USB.
  2. לחץ פעמיים על START_CHAT.bat.
  3. הדפדפן יפתח אוטומטית ב-http://localhost:5002.
  4. בקונסול תופיע גם כתובת הרשת — שתף עם התלמידים.
ℹ️ ה-launcher מנסה אוטומטית פורטים 5002, 5003, 5001, 8000, 8888, 5000 עד שנמצא פורט פנוי.
⚠️ אל תזיז ואל תשנה שמות קבצים בתיקייה המחולצת. השרת משתמש בנתיבים יחסיים.

📝 פורמט בסיס הידע

מקם קבצי .txt או .html בתיקיית knowledge_base/. TeacherBot קורא אותם בעת ההפעלה.

פורמט TXT (מומלץ)

1. RAM (זיכרון אקראי) — זיכרון זמני של המחשב המשמש בזמן הפעלת תוכניות.
2. מעבד (CPU) — הרכיב המרכזי המבצע הוראות תוכניות.
3. חומת אש — מערכת אבטחת רשת השולטת בתעבורה נכנסת ויוצאת.

פורמט HTML

השתמש בתגיות פשוטות בלבד. הבוט מחלץ טקסט גלוי, מתעלם מ-CSS וסקריפטים.

<p>חומת אש — מערכת לסינון תעבורת רשת.</p>
<p>כתובת IP — כתובת לוגית של התקן ברשת.</p>

המלצות

  • שמור קבצים בקידוד UTF-8 (חיוני לתווי עברית).
  • כתוב כל מונח פעם אחת בקובץ.
  • השתמש בפורמט מונח — הגדרה.
  • סיים משפטים בנקודה לתוצאת הקשר נקייה.

🔧 פתרון בעיות

בעיהפתרון
תלמידים לא יכולים להתחברודא שחומת האש של Windows אפשרה גישה לשרת. כל המכשירים חייבים להיות באותה רשת. נסה ping <IP המורה> ממכשיר תלמיד.
הדפדפן לא נפתח אוטומטיתפתח ידנית: http://localhost:5002 (אופליין) או http://localhost:5002 (מקוון).
השרת עולה אך יש שגיאותבדוק את חלון הקונסול. הסיבה הנפוצה: פורט תפוס. ה-launcher ימצא פורט פנוי אוטומטית.
TeacherBot מחזיר "לא נמצא"ודא שבתיקיית knowledge_base/ יש קבצי .txt או .html שמורים ב-UTF-8.
תווים פגומים / בעיות קידודשמור מחדש את קבצי בסיס הידע ב-UTF-8 (ללא BOM עבור .txt).
לא ניתן להתחבר — סיסמה שגויההמשתמשים שמורים ב-users_db.json. בקש מהמנהל למחוק וליצור מחדש את החשבון.
